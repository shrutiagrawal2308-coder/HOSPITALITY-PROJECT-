# AtliQ Grands Hospitality Analysis | Power BI

## 📌 Project Overview
This project is an interactive Power BI dashboard built using the **AtliQ Grands Hospitality Dataset**. The dashboard analyzes hotel booking and revenue data to provide insights into business performance, occupancy trends, booking platforms, cancellations, and key hospitality KPIs.

## 🎯 Objective
To help AtliQ Grands monitor business performance and make data-driven decisions by analyzing:
- Revenue performance
- Occupancy trends
- Booking platforms
- Room category performance
- Cancellation rates
- ADR and RevPAR

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📊 KPIs
- Revenue
- Total Bookings
- Occupancy %
- ADR
- RevPAR
- Realisation %
- Cancellation %

## 📂 Dataset
This project uses the **AtliQ Grands Hospitality Dataset** from the Codebasics Power BI Challenge. The dataset is intended for educational and portfolio purposes.

## 📈 Dashboard Features
- Revenue Analysis
- Occupancy Analysis
- Booking Platform Analysis
- City-wise Performance
- Room Category Analysis
- Cancellation Insights
- Interactive Filters & Slicers

## 💡 Key Insights
- Identified high-performing cities and hotels.
- Compared booking performance across platforms.
- Analyzed occupancy and revenue trends.
- Evaluated cancellation rates to identify improvement opportunities.
- Built an interactive dashboard for business decision-making.
